Theme System Design Overview

6 master colors → auto-generated shades → CSS variables → applied everywhere

How It Works

🎨
1. Admin Picks
6 color pickers in Settings
→
⚙️
2. Auto-Generate
HSL math creates 50-950 shades
→
💾
3. Cache & Store
Settings table + 1hr cache
→
🖥️
4. Inject CSS Vars
:root variables in <head>

The 6 Master Colors

Primary
#9A7B4F — Buttons, links, CTAs, active states
Secondary
#A67B5B — Gradients, badges, accents
Background
#FDFCFA — Page body, cards, surfaces
Heading
#1A1714 — h1-h6, section titles
Text
#4A433C — Body paragraphs, descriptions
Dark/Footer
#1A1714 — Footer bg, dark sections

CSS Variable Mapping

/* Injected in <head> from cached DB settings */
:root {
  --primary: 154 123 79;   /* RGB format for opacity support */
  --primary-50: 245 240 232;
  --primary-100: 232 220 200;
  /* ... 200-400, 600-950 ... */
  --secondary: 166 123 91;
  --bg: 253 252 250;
  --heading: 26 23 20;
  --text: 74 67 60;
  --dark: 26 23 20;
}

/* Tailwind uses them like: */
.bg-primary { background: rgb(var(--primary)); }
.bg-primary/10 { background: rgb(var(--primary) / 0.1); }
.text-primary { color: rgb(var(--primary)); }
