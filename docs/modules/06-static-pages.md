- Desktop: Moon/Sun icon in navbar
- Mobile: "More" menu toggle

### Color Scheme

| Element | Light Mode | Dark Mode |
|---------|-----------|-----------|
| Background | white (#ffffff) | #08090a |
| Text | gray-900 | gray-100 |
| Cards | white | gray-900 |
| Borders | gray-200 | white/5 |
| Primary | blue-500 | blue-400 |
| Hero gradient | blue-400 to blue-600 | blue-600 to blue-800 |

### Implementation
```css
/* Light mode (default) */
:root {
  --bg-primary: #ffffff;
  --text-primary: #111827;
  --border-color: #e5e7eb;
}

/* Dark mode */
.dark {
  --bg-primary: #08090a;
  --text-primary: #f3f4f6;
  --border-color: rgba(255, 255, 255, 0.05);
}
```

## 10. Copy Protection (CSS)

```css
/* Apply to content elements */
.copy-protected {
  user-select: none;
  -webkit-user-select: none;
  -moz-user-select: none;
  -ms-user-select: none;
}

/* Allow selection in specific areas */
.allow-select {
  user-select: text;
  -webkit-user-select: text;
}
```

Applied to 89.6% of page content to prevent scraping.
