# Module: User Accounts

## Overview
User authentication, profile management, and account settings. Supports OAuth (Google) and password-based authentication.

---

## URL Structure

```
/account/profile       - Profile settings
/account/contributions - User's edit contributions
/account/bookmarks     - Saved peptides
/account/connections   - Linked OAuth accounts
/account/preferences   - Notification preferences
```

---

## Authentication

### Sign-In Methods

1. **Google OAuth** (Primary)
   - One-click sign in with Google
   - Auto-creates account on first sign-in

2. **Password** (Secondary)
   - Can be added after OAuth sign-up
   - Requires at least one auth method

## API Endpoints

```typescript
// Auth
POST /api/auth/google          // Google OAuth initiate
GET  /api/auth/callback/google // OAuth callback
POST /api/auth/password        // Password sign-in
POST /api/auth/signup          // Password sign-up
POST /api/auth/signout         // Sign out

// Profile
GET  /api/user/profile         // Get profile
PUT  /api/user/profile         // Update profile

// Contributions
GET  /api/user/contributions   // List contributions
GET  /api/user/contributions/:id // Get single contribution

// Bookmarks
GET    /api/user/bookmarks     // List bookmarks
POST   /api/user/bookmarks     // Add bookmark
DELETE /api/user/bookmarks/:id // Remove bookmark

// Connections
GET    /api/user/connections   // List linked accounts
POST   /api/user/connections/password // Add password
DELETE /api/user/connections/:id // Unlink account

// Preferences
GET  /api/user/preferences     // Get preferences
PUT  /api/user/preferences     // Update preferences
```
