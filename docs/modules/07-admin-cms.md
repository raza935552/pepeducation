## Admin Authentication

Separate from user auth. Options:
1. Same NextAuth but with role check
2. Separate admin application
3. Use admin-specific OAuth (Google Workspace)

Recommended: Same auth system with role-based access control (RBAC).

```typescript
// Middleware example
function requireAdmin(handler) {
  return async (req, res) => {
    const session = await getSession(req);
    if (!session || !['admin', 'moderator', 'super_admin'].includes(session.user.role)) {
      return res.status(403).json({ error: 'Forbidden' });
    }
    return handler(req, res);
  };
}
```
